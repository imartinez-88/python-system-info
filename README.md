# Python System Info API

A clean, minimalist DevOps scaffold using **Flask** and **Docker**. This project demonstrates a production-ready repository structure with integrated CI/CD.

## Features
* **REST API**: Returns system health and CPU/RAM metrics.
* **Dockerized**: Includes a multi-stage Dockerfile for easy deployment.
* **CI/CD**: GitHub Actions automatically tests the code on every push.

## Tech Stack
* **Language**: Python 3.9
* **Framework**: Flask
* **Automation**: GitHub Actions
* **Containerization**: Docker
